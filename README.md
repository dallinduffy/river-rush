# 🚣 River Rush

A browser-based kayaking adventure game — paddle through 5 wild worlds, dodge obstacles, collect map pieces, and defeat each world's boss.

**[▶ Play Now](https://YOUR-USERNAME.github.io/river-rush)**

---

## Worlds

| # | World | Obstacle | Boss |
|---|-------|----------|------|
| 1 | 🏔️ Alpine Mountain | Rocks & Drifting Logs | Avalanche |
| 2 | 🌿 Jungle River | Whirlpools | Giant Croc Gauntlet |
| 3 | 🏜️ Desert Canyon | Falling Boulders | Flash Flood |
| 4 | 🧊 Arctic River | Polar Bear Paws | Ice Dam Burst |
| 5 | 🌊 Coastal Storm | Jellyfish Fields | Storm Surge |

## How to Play

- **Arrow keys** to steer
- **Space** to start / restart
- **ESC** to return to world select
- **Keys 1–5** to jump directly to a world (from the menu)

## Progression

Each world unlocks after clearing the previous one. Best scores are saved locally in your browser.

## Running Locally

All worlds are self-contained HTML files — no build step, no dependencies.

```bash
git clone https://github.com/YOUR-USERNAME/river-rush
cd river-rush
open index.html   # macOS
# or just double-click index.html in your file explorer
```
